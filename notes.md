# Projeto stack MERN

## Iniciar o projeto:
    npm init

## Instalar o express:
    npm i express

## Instalar o nodemon:
    npm i nodemon

No arquivo package.json, colocar um novo script:
    "dev": "nodemon index"

## Banco de dados
Na pasta ./src/database, criar o arquivo "db.js".
